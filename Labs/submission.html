Students should submit their code along with a README and videos.

For full credit, code should compile and run out of the box without editing.

The README can be very brief, consisting of

  • answers to questions (if applicable),
  • descriptions of what was completed, including extra challenges, and
  • how long the assignment took you and what was the hardest part

Videos/gifs should show results, especially any extra credit. Exceptional videos will be collected together and shown during class presentations. Please submit an ogv or mp4 of duration no more than 60 seconds.

Please place your videos or gifs in the same directory with the source.

WARNING! Files larger than 100 MB cannot be checked into git.

How to capture video using recordmydesktop

To restrict the recording to just your homework window, you need to pass the window id to recordmydesktop. The window id can be obtained with xwininfo. Type Ctrl-C to stop recording.

  • Get the window id: Run xwininfo
  • Capture your demo without sound: Run recordmydesktop --windowid [WindowId] --no-sound --v_quality 30 -o demo.ogv
  • Check the video: Run mplayer demo.ogv or run firefox demo.ogv
